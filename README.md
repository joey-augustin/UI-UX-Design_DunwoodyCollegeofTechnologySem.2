# UI-UX-Design_DunwoodyCollegeofTechnologySem.2
Hands-on UI/UX projects emphasizing user research, wireframing, prototyping, and responsive web design with Figma.

## Project
I designed a website for a company idea, _Manna Coffee Cart_, a coffee shop/food truck hybrid website with functions like viewing a menu, ordering, creating a perks account, and buying merchandise.

- **Low-Fidelity Prototype (Desktop)**: I learned the importance of user flow and website functionality. I had the opportunity to experiment with different layouts to create the highest level of usability.
